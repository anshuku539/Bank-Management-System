Bank Management System

Console-based C++ project. Build and run locally (instructions in README.md).

View on GitHub

About

This repository contains bank_system.cpp, a simple C++ bank system for learning and demonstration purposes.

Build

g++ -std=c++17 -Wall -Wextra -O2 "bank_system.cpp" -o "bank_system.exe"

Run

& "./bank_system.exe"

Notes
